üß™ Testing Guide

Step-by-Step Testing Process

  1. Connect Shop: Use OAuth to connect your Shopify store first
  2. Create Subscription: Use the form above to create a billing subscription
  3. Approve Subscription: Visit the confirmation URL from the response
  4. Get Line Item ID: Copy the subscription line item ID from the response
  5. Create Usage Records: Test charging fees using the usage records form
  6. Monitor in Shopify: Check your Shopify Partner Dashboard for billing data

üìö Key Concepts

  • App Subscription: The billing container with a capped amount
  • Usage Records: Individual charges for orders/services
  • Capped Amount: Maximum billing in a period (prevents runaway charges)
  • Confirmation URL: Merchant approves billing in Shopify admin
